================================================================================
KEY FINDINGS
================================================================================

1. SIMPLE MODELS WIN: Linear Regression with engineered features achieves the
   best performance (RMSE=0.0412, MAPE=1.18%), outperforming all complex models.
   This suggests gasoline prices are strongly autocorrelated and well-captured
   by linear combinations of lagged/rolling features.

2. FEATURE ENGINEERING MATTERS: Adding rolling statistics, momentum, and date
   features reduces Linear Regression RMSE by 38% compared to lag-only features.

3. ALL MODELS BEAT BASELINE: Every trained model outperforms the naive baseline
   (predict last week's price), confirming that ML adds value to this task.

4. NEURAL NETWORKS UNDERPERFORM: With only ~1300 training samples, neural
   networks (MLP, LSTM, GRU) cannot outperform simpler models. The best LSTM
   (RMSE=0.0921) is 2x worse than Linear Regression. This aligns with the
   common finding that deep learning needs large datasets to shine.

5. LSTM > MLP: Among neural networks, sequence-based models (LSTM, GRU) clearly
   outperform feedforward MLPs, suggesting temporal structure is important.
   GRU matches LSTM with fewer parameters.

6. TREE MODELS ARE COMPETITIVE: XGBoost (RMSE=0.0772) slightly outperforms
   Random Forest and matches the naive baseline. Feature importance reveals
   lag-1 price is by far the most important feature.

7. LONGER HORIZONS ARE HARDER: 4-week prediction RMSE is typically 2-3x worse
   than 1-week, with MAPE increasing from ~1-3% to ~4-9%. This is expected
   for time series with inherent uncertainty.

8. COMPLEXITY ≠ ACCURACY: For this dataset size and problem, model complexity
   does not correlate with accuracy. The simplest model (Linear Regression)
   performs best, while the most complex (MLP-Large, deep LSTM) perform worst.
   This is an important practical lesson in ML model selection.
