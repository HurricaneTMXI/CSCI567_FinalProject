# CSCI 567 Final Project: Weekly Gasoline Price Forecasting

A comparative study of machine learning methods for predicting weekly U.S. gasoline prices.

**Team:** Simou Chen, Khalid Ali, Mario Prado, Jasper Fan-Chiang

## How to Run

**Notebooks 01, 02, 03, 05** run locally:
```bash
cd notebooks/
jupyter notebook
```
Run in order: 01 → 02 → 03 → 05.

**Notebook 04** runs on Google Colab (requires PyTorch + GPU):
1. Upload `04_neural_network_models.ipynb` to Google Colab
2. Set runtime to GPU (Runtime → Change runtime type → T4 GPU)
3. Run all cells — upload `PET_PRI_GND_DCUS_NUS_W.csv` when prompted
4. Download the result JSON files from the Colab `results/` folder
5. Copy them into the local `results/` directory
6. Re-run notebook 05 locally to include neural network results in the comparison
